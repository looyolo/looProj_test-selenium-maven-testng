
【1】Selenium 的身世:

2004 年，Selenium 1 = Selenium Core = Selenium RC（Remote Control） + Selenium IDE ，

    基于 JS 开发，开源作者 ThoughtWorks 公司的 Jason Huggins 。

2006~2008 年，Selenium 2 = Selenium 1 + Selenium WebDriver ，

    贡献者 Google 公司的 Simon Stewart ，启动了 WebDriver 项目。

2016 年，Selenium 3 = Selenium 2 - Selenium RC + Selenium Grid ，

    实现了 核心 API 与 Client Driver 的分离，不再自带浏览器。
    Firefox 必须 48 以上，IE 必须 9.0 以上。
    只支持 JDK 1.8 及以上版本。


【2】学习 Selenium 工具的能力要求是什么？

（1）需要更多的知识和编程技能。

常常遇到的一些知识领域：HTML, CSS, JS, XML, J/Py, MySQL, JUnit/TestNG, Ant/Maven, Jenkins, Cucumber 等。

（1）需要更多的编程实践积累的代码量。

涉及到编程的，编程能力的高低，直接决定你是否可以驾驭优秀的开发/测试框架。
真正的自动化测试高手，从技术能力上来说，比中级开发的水平还要高。
